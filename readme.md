## 🎯 About This Repo

This is my structured Linux learning journey — built as part of my path toward **Security Engineering / DevSecOps**. Every command I learn, every concept I struggle with, every mini-project I build ends up here.

I'm not just collecting tutorials — I'm building a reference I'll come back to during HackTheBox labs, DevOps work, and real-world incident response.

## 🧠 What I've Learned So Far

### Day 1 ✅
- Shell vs terminal (they're not the same thing)
- Filesystem hierarchy (`/`, `/etc`, `/var`, `/home`)
- Navigation: `pwd`, `cd`, `ls`
- Absolute vs relative paths
- File creation: `touch`, `file`
- `echo`, variables, command substitution `$()`
- Arithmetic with `$(( ))`
- Reading user input with `read`
